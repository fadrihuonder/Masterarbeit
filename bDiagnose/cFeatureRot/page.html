menu Diagnose

Features werden in der Feature-Liste rot markiert

Ein Feature (z.B. Linear austragen) wird in der Feature-Liste rot markiert.

Zudem wird es im Modell nicht umgesetzt.

> Fehler, Möglichkeit 1:

Das Feature wurde falsch konfiguriert

Ein Fehler könnte sein, dass du bei der Konfiguration des Features einen Fehler gemacht hast.

arrow_downward

> Lösung:

Veränderungen rückgängig machen

Prüfe nach, ob du alle notwendigen Schritte korrekt ausgeführt hast.

Suche dafür im Bereich Anleitungen die Anleitung für das gewünschte Feature.

> Fehler, Möglichkeit 2:

Veränderung an übergeordneten Feature

Dein Feature bezieht sich auf vorausgehende Features (z.B. eine frühere Skizze) oder einzelne Instanzen solcher Features. Ein möglicher Fehler besteht darin, dass du weiter oben in der Feature-Liste etwas verändert hast.

Das führt dazu, dass für das gestörte Feature wichtige Bezugsgrösse verändert oder gelöscht wurden: Das Feature kann nicht mehr aufelöst, berechnet und dargestellt werden.

arrow_downward

> Lösung:

Veränderungen rückgängig machen

Führe das übergeordnete Feature wieder in den alten Zustand zurück.

Am einfachsten gelingt das dir das mit dem "Rückgängig"-Pfeil oben links.

Feature anpassen

Du kannst das gestörte Feature auch an die neuen Umstände anpassen: Mit einem Doppelklick auf das Feature (in der Feature-Liste) erscheint wieder das übliche Dialogfeld des Features.
